## ğŸ¯ Vision

Konekt is an all-in-one event management platform designed for Czech and Slovak startup and business events with 30-150 participants. We're eliminating tool fragmentation that forces organizers to juggle multiple applications and attendees to download different apps for each event.

**Our goal:** Make event organization effortless and event participation seamless.

---

## ğŸ” The Problem

### For Event Organizers:
- **Tool Chaos:** Managing one event requires 4-5 different tools
  - Eventee (program/agenda)
  - Excel/Google Sheets (attendee tracking, check-in)
  - WhatsApp/Telegram (communication)
  - Notion (internal notes, planning)
  - Zoom/Google Meet (if hybrid)
- **Time Waste:** 5+ hours per event just coordinating tools
- **Data Fragmentation:** Participant info scattered across platforms
- **Check-in Nightmare:** Paper lists or complex QR systems from expensive platforms
- **No Unified View:** Can't see event health at a glance

### For Event Attendees:
- **App Overload:** Different app for every event attended
- **Lost Information:** Event details buried in email threads
- **Poor Networking:** No visibility into who's attending before/during event
- **Friction:** Registration in one place, program in another, networking elsewhere

### Real-World Experience:
Having attended events like BeNextOne, Innovation Week, and other Prague startup gatherings, we've lived this frustration firsthand. Every event = new app to download, new login to create, new interface to learn.

## ğŸ¨ Core Philosophy

### 1. **Simplicity Over Features**
We don't build everything - we build what matters. No bloated features, no unnecessary complexity.

### 2. **Modular by Design**
Every feature is a self-contained module. Need QR check-in but not networking? Fine. Want custom branding? Easy.

### 3. **Mobile-First for Attendees, Desktop-First for Organizers**
- Attendees use the platform during events (on phones)
- Organizers plan events at desks (on computers)

### 4. **Integration, Not Replacement**
We don't replace LinkedIn for networking or WhatsApp for messaging. We bridge to existing tools people already use and trust.

### 5. **Data Clarity**
Every metric is visible, every export is simple. Organizers own their data.
